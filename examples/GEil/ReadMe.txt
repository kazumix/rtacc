========================================================================
    GEil サンプル（GEtest.IL）
========================================================================

GEtest.IL を LLVM IR に変換し、main.c から周期呼び出しする最小例です。

  - GEtest.IL … IL ソース（VAR / LD / ST / CAL など）
  - main.c … GEtest() をループで呼び出し

ビルド・実行手順はプロジェクトのルート README または project.json を参照してください。
